# Quantum Computing Personal Projects

## Overview
This repository is a personal collection of projects in quantum computing, combining theoretical study and practical implementation.

Theoretical notes are developed in Spanish, reflecting my own understanding of the material, while all code and technical documentation are written in English to ensure clarity and accessibility.

## Objective
The main goal of this repository is to support my long-term professional development in quantum computing through a structured and self-driven learning process.

Each project is approached from first principles, with an emphasis on:
- Building a solid theoretical foundation
- Developing reproducible implementations
- Documenting progress in a clear and organized manner

## Current Focus
I am currently studying **Annealing**, with a particular focus on:

- **OceanSDK**
- **Simulated Annealing**
- **Quantum Annealing**

The objective is to develop a new key understanding of the Quantum Annealing Framework for working in Quantum Computing models

## Teaching Context
This work is also aligned with my role as a Teaching Assistant, with the intention of:
- Strengthening my own understanding through structured study
- Developing clear and didactic explanations
- Building material that can be used for teaching and mentoring

## Structure
Each project in this repository typically includes:
- **Notes**: Theoretical development and conceptual understanding (in Spanish)
- **Experiments**: Jupyter notebooks with implementations and simulations. For Annealing, I decided to merge the notes with the experiments
- **Logs**: A record of progress, insights, and challenges
- **References**: Sources used throughout the study process
